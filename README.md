# Smart Home Data Pipeline 🏠

Полноценный ETL-пайплайн для умного дома с 1400 датчиками.

## Архитектура
- 🐍 **Producer**: асинхронный генератор данных (aio-pika, asyncio)
- 🐰 **RabbitMQ**: брокер сообщений
- 🐼 **Consumer**: забор данных, сохранение в Parquet
- 🔥 **Spark**: обработка, удаление дубликатов. Версия 3.5.0.
- 🏢 **ClickHouse**: аналитическое хранилище. Версия 26.2.3.2.
- ☁️ **Airflow**: оркестрация DAG. Версия 2.8.1.
- 📊 **Superset**: для построения дашбордов
- 🐳 **Docker**: 10 контейнеров в связке


## Запуск
```bash
docker-compose up -d
```

## Необходимый файл
`clickhouse-spark-runtime-3.5_2.12-0.10.0.jar`

### Инструкция по установке:
1. Скачайте JAR-файл с [Maven Central](https://mvnrepository.com/artifact/com.clickhouse.spark/clickhouse-spark-runtime-3.5_2.12/0.10.0)
2. Создайте папку `jars/` в корне проекта и поместите туда скачанный файл

## Результат
- Датчики генерируют данные
- Данные летят в очередь RabbitMQ, consumer разбирает очередь, формирует батчи, преобразует их в формат Parquet и отправляет их на обработку в Spark
- Spark обрабатывает данные и кладёт в Clickhouse
- 1000+ записей в ClickHouse за час
- Подключен Superset, в котором можно строить дашборды
